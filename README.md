# Enterprise Banking Data Platform

## SQL Server 2019 to Microsoft Fabric Migration
###  Banking Colud Data Modernization Project

---

## Project Overview

Enterprise Banking Data Platform is a data modernization project that demonstrates migration of banking workloads from SQL Server 2019 to Microsoft Fabric Lakehouse architecture.

The project covers enterprise banking data engineering practices including data modeling, ETL pipelines, data transformation, analytics, and reporting.
